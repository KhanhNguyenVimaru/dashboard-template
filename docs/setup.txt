HƯỚNG DẪN TẢI VÀ CÀI ĐẶT DỰ ÁN distributed-iot-routing-simulator (SỬ DỤNG MySQL)

1. TẢI CODE TỪ GITHUB

- Yêu cầu: Đã cài đặt Git trên máy tính.
- Mở Terminal (hoặc CMD, PowerShell).
- Chạy lệnh sau để clone code về máy:

git clone https://github.com/KhanhNguyenVimaru/distributed-iot-routing-simulator.git

- Di chuyển vào thư mục dự án:

cd distributed-iot-routing-simulator

2. CÀI ĐẶT BACKEND (LARAVEL)

- Yêu cầu: PHP >= 8.2, Composer, MySQL.
- Cài đặt các package PHP:

composer install

- Tạo file cấu hình môi trường:

cp .env.example .env

- Tạo database MySQL mới, ví dụ: distributed_iot

- Mở file .env, chỉnh sửa các thông số kết nối MySQL:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=distributed_iot
DB_USERNAME=your_mysql_user
DB_PASSWORD=your_mysql_password

- Tạo key cho ứng dụng:

php artisan key:generate

- Chạy migrate để tạo bảng:

php artisan migrate

- (Tùy chọn) Seed dữ liệu mẫu:

php artisan db:seed

3. CÀI ĐẶT FRONTEND (VITE + VUE)

- Yêu cầu: Node.js >= 20, npm.
- Di chuyển vào thư mục web:

cd web

- Cài đặt các package JS:

npm install

- Chạy ứng dụng frontend (dev):

npm run dev

4. CHẠY ỨNG DỤNG

- Ở thư mục gốc, chạy server Laravel:

php artisan serve

- Backend chạy tại địa chỉ: http://localhost:8000

- Nếu dùng cả frontend riêng (web/), truy cập http://localhost:5173 để phát triển giao diện.

5. GHI CHÚ

- Nếu gặp lỗi về quyền truy cập file, hãy đảm bảo các thư mục `storage/` và `bootstrap/cache/` có quyền ghi.
- Để build frontend cho production:
  Vào thư mục `web/` và chạy:
  npm run build
