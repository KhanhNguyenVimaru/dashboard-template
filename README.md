# IoT Dashboard Template

This repository is a clean and modern **IoT Dashboard Template** that follows standard design and backend architecture practices.  
It provides a ready-to-use foundation for building IoT management systems, focusing on usability, maintainability, and scalability.

---

## Overview

The template includes:
- A **standardized modal design** with smooth transitions, validation, and reusable structure.
- **Databox and chart components** for displaying key IoT metrics clearly and efficiently.
- A **backend** implementing full **CRUD** operations, **search** functionality, and **RESTful API** endpoints written with proper structure and conventions.

---

## Frontend Features

- Responsive dashboard layout with sidebar and top navigation.
- Modular component system for modals, databoxes, and charts.
- Unified styling using Tailwind CSS.
- Chart integration (Chart.js) for real-time visualization.
- Focused on clarity, speed, and reusability.

---

## Backend Features

- Full CRUD API for managing devices, sensors, and related entities.
- Search and filter endpoints for efficient data handling.
- RESTful design with consistent routes, status codes, and response structure.
- Built with clean controller-service-model architecture.
- Easily extendable for IoT data collection or MQTT integration.
